# 下位机固件任务计划

## 状态定义

- `[基线]`：软件实现已经存在，并通过构建或静态检查。
- `[待板测]`：软件实现已经存在，但仍需硬件验证。
- `[待实现]`：尚未实现。
- `[待确认]`：缺少硬件身份或额定参数证据，确认前不得代入参考值。
- `[待决策]`：候选和证据尚未完成统一评审，不得写成冻结方案。

## 已确认事实与未冻结候选

- MCU 当前运行在 168 MHz；不通过调整 SYSCLK 改变控制周期。
- PWM 当前约 20 kHz，速度任务当前为 10 ms（100 Hz）；两者是现状基线，不是最终冻结值。
- PWM 计数范围当前为 -4200 至 +4200；最终调制方式和限幅随 G7 决策。
- 编码器分辨率：电机轴四倍频后 1024 计数，减速比 1:30，每轮 30720 计数。
- 首个闭环验证对象：MA 电机。
- 当前硬件没有 MCU 可用的电机电流反馈，禁止实现伪电流环。
- 速度环以 1 kHz、机体环以 100-200 Hz 作为首轮实验候选；最终频率由 G7 决策。
- PI + 前馈是基准控制器，LADRC 和 PI + DOB 是候选控制器，尚未拍板。
- M0-M3 不包含麦轮运动学和二进制 ROS 协议。
